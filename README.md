# NovaReader — современный аналог DjVuReader

NovaReader — это современный reader с упором на практичность: больше инструментов в ежедневной работе с документами и понятный интерфейс.

## Что реализовано сейчас

- Современный интерфейс: библиотека + расширенный toolbar + центральная область просмотра.
- Поддержка форматов в текущей версии:
  - PDF (рендер страниц через PDF.js)
  - Изображения (PNG, JPG, WEBP, GIF, BMP)
- Инструменты чтения и навигации:
  - переходы между страницами и переход к конкретной странице
  - масштаб, fit-to-width, fit-to-page
  - поворот страницы
  - полноэкранный режим
  - колёсико + Ctrl для масштабирования
  - drag&drop открытия файла
- Работа с содержимым:
  - поиск по PDF
  - переход по совпадениям (вперёд/назад)
  - извлечение текста текущей страницы + copy/export в TXT
- Персонализация и продуктивность:
  - локальная библиотека последних открытых файлов
  - заметки к документу + экспорт заметок в `.txt`
  - закладки по страницам (добавить/перейти/очистить)
  - оглавление PDF с переходом по разделам
  - информация о файле (формат, размер, количество страниц)
  - скачать исходный файл и печать текущей страницы
- Тёмная/светлая тема и горячие клавиши.

## Запуск

```bash
python3 -m http.server 4173
```

Открыть в браузере:

```text
http://localhost:4173/app/
```
