# NovaReader — современный аналог DjVuReader

NovaReader — это современный reader с упором на практичность: больше инструментов в ежедневной работе с документами и понятный интерфейс.

## Что реализовано сейчас

- Современный интерфейс: библиотека + расширенный toolbar + центральная область просмотра.
- Поддержка форматов в текущей версии:
  - PDF (рендер страниц через PDF.js)
  - Изображения (PNG, JPG, WEBP, GIF, BMP)
- Инструменты чтения и навигации:
  - переходы между страницами и переход к конкретной странице
  - масштаб, fit-to-width, fit-to-page
  - поворот страницы
  - полноэкранный режим
  - колёсико + Ctrl для масштабирования
  - drag&drop открытия файла
- Работа с содержимым:
  - поиск по PDF/DjVu (для DjVu через импорт data JSON: текстовый слой/оглавление/изображения страниц)
  - переход по совпадениям (вперёд/назад)
  - извлечение текста текущей страницы + copy/export в TXT
- Персонализация и продуктивность:
  - Аннотации на странице: перо/маркер/ластик, undo, очистка, экспорт PNG, JSON для страницы и bundle JSON для всего документа и комментарии-пины на странице и фигуры (прямоугольник/стрелка/линия/окружность)
  - локальная библиотека последних открытых файлов (удаление записей и быстрая очистка списка)
  - заметки к документу (заголовок/теги/текст) + автосохранение при вводе, индикатор статуса сохранения, экспорт в `.txt`/`.md`/`.json`, импорт из `.json` с режимами replace/append и быстрый timestamp
  - закладки по страницам (добавить/перейти/очистить) + импорт/экспорт `.json` и фильтр по названию/странице
  - оглавление PDF с переходом по разделам
  - быстрые превью страниц (миниатюры) с переходом в один клик
  - информация о файле (формат, размер, количество страниц)
  - скачать исходный файл и печать текущей страницы
  - экспорт/импорт workspace backup (`.json`) для текущего документа (заметки, закладки, hotkeys, тема, аннотации, комментарии)
  - Stage 4: OCR import `.json` (pagesText), cloud push/pull workspace по URL endpoint, и локальная collaboration-синхронизация через BroadcastChannel
  - автосохранение позиции чтения (страница/масштаб/поворот) и быстрый сброс прогресса
  - история навигации по страницам (назад/вперёд) с кнопками и Alt+←/Alt+→
  - список недавних переходов по страницам (быстрое возвращение к последним точкам чтения)
  - список активных результатов поиска по страницам с быстрым переходом к каждому совпадению и счётчиком совпадений на странице + экспорт в `.json`/`.csv`, импорт результатов из `.json`/`.csv`, экспорт summary в `.txt` и копирование summary в буфер
  - история поисковых запросов (быстрый повтор) + очистка, экспорт `.json`/`.txt`, копирование и импорт `.json`
  - переключатель области поиска: весь документ или только текущая страница
  - живая статистика документа: количество аннотаций/комментариев/закладок и темп чтения (стр./ч)
  - учёт времени чтения по документу (автотрекинг активного окна) и ручной сброс таймера
  - ETA завершения чтения (прогноз даты/времени окончания на основе текущего темпа)
  - цель чтения по странице с индикатором остатка и ETA до выбранной цели
- Тёмная/светлая тема, настраиваемые горячие клавиши (с валидацией конфликтов, подсветкой ошибок, inline-подсказками и авто-фиксом) и встроенная подсказка Shortcuts.

## Запуск как обычное приложение (.exe, без Python и серверов)

NovaReader можно собирать и запускать как desktop-приложение на Windows 10/11.

1. Установить Node.js LTS (включая npm).
2. В корне проекта выполнить:

```bash
npm install
```

3. Собрать portable `.exe`:

```bash
npm run build:win:nosign
```

После сборки готовый файл будет в папке `dist/` (например, `NovaReader-1.0.0.exe`).

Для установщика (NSIS):

```bash
npm run build:win:installer
```

Для локального desktop-запуска без сборки:

```bash
npm run start:desktop
```

## Запуск

```bash
python3 dev_server.py
```

Открыть в браузере (можно сразу `/`, сервер перенаправит на `/app/`):

```text
http://localhost:4173/
```

Встроенные release endpoint'ы:

```text
http://localhost:4173/api/health
http://localhost:4173/api/workspace
```
