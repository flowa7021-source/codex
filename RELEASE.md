# RELEASE.md

## Release target
- Product: OfflineDocStudio
- Release: **1.0.0**

## Artifacts
1. ZIP package (CPack):
   - `OfflineDocStudio-1.0.0-<platform>.zip`
2. Windows installer (Inno Setup):
   - `dist/OfflineDocStudio-Setup-1.0.0.exe`

## Release commands (Linux/macOS)
```bash
cmake -S . -B build
cmake --build build -j
ctest --test-dir build --output-on-failure
cpack --config build/CPackConfig.cmake
```

## Release commands (Windows)
```powershell
./scripts/build_windows_installer.ps1
```

## Verification checklist
- Build and tests pass.
- `--doctor` reports healthy runtime layout after `--init-layout`.
- Interactive mode runs and exits cleanly.
- ZIP package is generated by CPack.
- Inno Setup script is up-to-date with current release version.
