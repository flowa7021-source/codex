# Продуктовый план: NovaReader

## 1) Цель продукта

Сделать современный аналог DjVuReader, который:
- быстрее осваивается новым пользователем;
- поддерживает больше сценариев работы с документами;
- имеет масштабируемую архитектуру для форматов (PDF, DjVu, изображения, ePub);
- работает кросс-платформенно (Web/Desktop).

## 2) Целевая аудитория

- Студенты и преподаватели.
- Инженеры и исследователи, работающие с технической документацией.
- Пользователи архивов со старыми сканами и DjVu/PDF.

## 3) Ключевые боли текущих решений

- Устаревший интерфейс и навигация.
- Мало инструментов для заметок и поиска.
- Слабая персонализация и отсутствие локальной библиотеки.

## 4) Ценностное предложение NovaReader

- Современный UI/UX (тёмная тема, горячие клавиши, адаптивность).
- Быстрый просмотр и навигация.
- Модульная система рендер-адаптеров под разные форматы.
- База для функций Pro: OCR, аннотации, облачная синхронизация.

## 5) Функциональный scope

### MVP (реализовано)
- Загрузка и чтение PDF.
- Загрузка и просмотр изображений.
- Панель инструментов: масштаб, fit-to-width, fit-to-page, поворот.
- Переходы между страницами и jump-to-page.
- Поиск по PDF + навигация по совпадениям.
- Оглавление PDF с переходом по структуре документа.
- Извлечение текста текущей страницы + экспорт.
- Локальная библиотека недавних файлов + drag&drop открытия.
- Заметки к документу + экспорт в TXT.

### Phase 2
- Полная поддержка DjVu (декодирование, текстовый слой, оглавление).
- Аннотации (выделение, комментарии, фигуры).
- Слои рендера для overlay-инструментов.

### Phase 3
- OCR-пайплайн.
- Синхронизация документов и заметок.
- Плагины и API интеграций.
