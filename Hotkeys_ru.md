# Hotkeys_ru.md

## Дефолтная карта хоткеев

- `Ctrl+O` — Открыть файл
- `Ctrl+W` — Закрыть вкладку
- `Ctrl+Tab` / `Ctrl+Shift+Tab` — Переключение вкладок
- `Ctrl+F` — Поиск
- `F11` — Полноэкранный режим
- `Ctrl+Plus` / `Ctrl+Minus` — Zoom in/out
- `Ctrl+0` — Fit page
- `Ctrl+1` — Actual size
- `Ctrl+Alt+X` — Screenshot (copy selection as image)
- `Ctrl+Shift+O` — OCR выделения
- `Space (hold)` — Временный Hand tool
- `Middle mouse drag` — Pan
- `Delete` — Удалить выбранные страницы (в Pages Overview)
- `Ctrl+R` — Поворот страницы 90°

## Требования
- Полная переназначаемость.
- Экспорт/импорт в JSON (`config/hotkeys.json`).
- Независимость от раскладки клавиатуры (физические scan-codes + fallback).
